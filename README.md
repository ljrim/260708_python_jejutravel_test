# 🏝️ 제주도 여행 일정 추천 & 상권 분석

제주상권 공공데이터를 활용해 **개인 페르소나별 여행 일정을 추천**하고,
**업종·동네별 상권을 분석**하는 Streamlit 웹앱입니다.

## ✨ 주요 기능

### 🗺️ 여행 일정 추천
- **페르소나 5종**(미식가 · 커플 · 가족 · 나홀로 · 액티비티 마니아) 선택 → 어울리는 여행 컨셉 추천
- **여행 컨셉 5종**: 🍜 맛집 미식 투어 · ☕ 감성 카페 투어 · 🏖️ 힐링 & 웰니스 · 🎨 문화 & 액티비티 · 🛍️ 쇼핑 & 기념품
- 지역·여행 일수·하루 방문 수를 고르면 **가까운 장소끼리 이어주는 최적 루트**로 일정 생성
- 일자별 일정표 + **방문 순서 숫자 마커와 이동 경로선이 표시된 지도**

### 📊 상권 분석
- **업종 소분류별 매장 수** 막대그래프
- **동네(행정동)별 매장 수** 막대그래프
- 선택한 업종의 **매장 위치 지도**

## 📁 파일 구성

| 파일 | 설명 | 배포 |
|------|------|:---:|
| `app.py` | 메인 Streamlit 앱 | ✅ |
| `jeju_data.csv.gz` | 경량 데이터 (약 1.7MB, UTF-8) | ✅ |
| `requirements.txt` | 설치할 파이썬 패키지 목록 | ✅ |
| `.streamlit/config.toml` | 앱 테마 설정 | ✅ |
| `prepare_data.py` | 원본 CSV → 경량 데이터 변환 스크립트 | ✅ |
| `README.md` | 프로젝트 설명 | ✅ |
| `제주상권.csv` / `*.xlsx` | 원본 대용량 데이터 (배포 제외, `.gitignore`) | ❌ |

> 원본 `제주상권.csv`(약 21MB)에서 앱이 쓰는 8개 컬럼만 추려
> `jeju_data.csv.gz`(약 1.7MB, **약 92% 감소**)로 압축했습니다.

## 🚀 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ Streamlit Cloud 배포 방법

1. 이 폴더의 파일들을 **GitHub 저장소**에 올립니다.
   (`.gitignore` 덕분에 대용량 원본 파일은 자동으로 제외됩니다.)
2. [share.streamlit.io](https://share.streamlit.io) 접속 → **New app**
3. 저장소·브랜치 선택, **Main file path**에 `app.py` 입력
4. **Deploy** 클릭 → 잠시 후 앱 URL이 생성됩니다.

## 🔄 데이터 갱신

원본 데이터가 새로 바뀌었을 때만 아래를 실행해 경량 파일을 다시 만듭니다.

```bash
python prepare_data.py
```
